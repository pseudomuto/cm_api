# Cloudera Manager API (cm_api) Ruby Client

Work in progress...don't even try it out just yet.

## Installation

Add this line to your application's Gemfile:

```ruby
gem "cm_api"
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install cm_api
